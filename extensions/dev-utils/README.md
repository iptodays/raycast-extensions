# Dev Utils

A Raycast extension with everyday developer utilities.

## Commands

| Command | Description |
|---------|-------------|
| **JSON Tool** | Format, minify, validate, escape / unescape JSON strings |
| **Base64 Tool** | Encode and decode Base64 strings |
| **UUID Tool** | Generate UUID v4 / v7, validate UUID strings |
| **Timestamp Tool** | Convert between Unix timestamps and readable dates, get current time |
| **URL Tool** | URL encode / decode, parse URL components and query parameters |
| **Color Tool** | Convert between HEX, RGB, and HSL color formats |

## Usage

Open Raycast and search for any command name listed above.
